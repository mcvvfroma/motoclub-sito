# Motoclub VVF Roma - Portale Soci

Questo è il portale ufficiale della Sezione di Roma del Motoclub Vigili del Fuoco. 
L'applicazione è sviluppata in Next.js ed è progettata per gestire la vita associativa del club.

## Funzionalità principali

- **Calendario Eventi**: Gestione delle uscite ufficiali con integrazione meteo dinamica (Open-Meteo & ilMeteo.it).
- **Galleria Fotografica**: Caricamento di ricordi digitali direttamente dai dispositivi dei soci.
- **Comunicazioni**: Bacheca ufficiale del Direttivo per verbali e avvisi urgenti.
- **Area Riservata**: Accesso protetto per i soci con pannello amministrativo per la gestione dell'anagrafica.
- **Convenzioni**: Catalogo dei vantaggi e partner tecnici per i soci.

## Tecnologie utilizzate

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS & ShadCN UI
- **AI**: Google Gemini via Genkit per l'analisi delle condizioni di guida.
- **PWA**: Supporto per l'installazione su dispositivi mobili come Web App.
